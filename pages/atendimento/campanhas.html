Campanhas (Disparos API)

Envie mensagens em massa via API Oficial e Não Oficial do WhatsApp

O que são Campanhas?

Campanhas são disparos de mensagens do WhatsApp realizados através da API Oficial (WABA) e API Não Oficial. Permitem enviar mensagens para múltiplos contatos de forma automatizada, respeitando as políticas do WhatsApp.

✅

API Oficial (WABA)

Maior confiabilidade e recursos avançados

⚡

API Não Oficial

Mais flexível, ideal para testes

Como Utilizar

Passo 1: Acessar o Módulo

  1. No menu lateral, clique em Atendimento
  2. Selecione Campanhas
  3. Você verá a lista de campanhas existentes

Passo 2: Criar Nova Campanha

  1. Clique no botão "Nova Campanha"
  2. Preencha os dados básicos:
    • Nome da Campanha: Identificação interna
    • Descrição: Objetivo da campanha
    • Canal: Selecione o número do WhatsApp
  3. Clique em "Avançar"

Passo 3: Selecionar Destinatários

Você pode escolher os destinatários de 3 formas:

  • Lista de Contatos: Selecione uma lista pré-cadastrada
  • Importar CSV: Faça upload de arquivo com números
  • Manual: Digite os números manualmente

💡 Formato do CSV

O arquivo CSV deve conter uma coluna com os números no formato:

  • 5511999999999 (com DDI e DDD)
  • Sem espaços, traços ou parênteses
  • Um número por linha

Passo 4: Criar a Mensagem

Para API Oficial (WABA):

  1. Selecione um Template Aprovado pela Meta
  2. Preencha as variáveis do template (se houver)
  3. Visualize a prévia da mensagem

⚠️ Importante

Na API Oficial, você só pode usar templates previamente aprovados pela Meta. Não é possível enviar mensagens livres.

Para API Não Oficial:

  1. Digite a mensagem livremente
  2. Adicione variáveis personalizadas (opcional):
    • {{nome}} - Nome do contato
    • {{empresa}} - Nome da empresa
    • {{custom1}} - Campo personalizado
  3. Adicione mídias (imagem, vídeo, documento)

Passo 5: Configurar Envio

  • Envio Imediato: Dispara assim que confirmar
  • Agendar: Escolha data e hora para envio
  • Intervalo entre mensagens: Defina delay (recomendado: 3-5 segundos)
  • Horário de envio: Respeite horário comercial (8h-20h)

💡 Boas Práticas

  • Evite enviar fora do horário comercial
  • Use intervalo de pelo menos 3 segundos entre mensagens
  • Teste com poucos contatos antes de enviar para todos
  • Sempre ofereça opção de descadastro

Passo 6: Revisar e Enviar

  1. Revise todos os dados da campanha
  2. Verifique a quantidade de destinatários
  3. Confira a mensagem e variáveis
  4. Clique em "Confirmar e Enviar"

Acompanhamento da Campanha

Status da Campanha

  • Aguardando: Campanha agendada, aguardando horário
  • Em Andamento: Mensagens sendo enviadas
  • Concluída: Todos os envios finalizados
  • Pausada: Campanha pausada manualmente
  • Cancelada: Campanha cancelada antes de concluir

Métricas Disponíveis

  • Total de Envios: Quantidade de mensagens enviadas
  • Entregues: Mensagens que chegaram ao destinatário
  • Lidas: Mensagens visualizadas (apenas API Oficial)
  • Falhas: Mensagens que não foram entregues
  • Taxa de Entrega: Percentual de sucesso

Recursos Avançados

Personalização com Variáveis

Use variáveis para personalizar cada mensagem:

  • {{nome}} - Nome do contato
  • {{empresa}} - Nome da empresa
  • {{email}} - Email do contato
  • {{custom1}} a {{custom5}} - Campos personalizados

Exemplo:

Olá {{nome}}! 

Temos uma oferta especial para {{empresa}}.

Acesse: www.exemplo.com/{{custom1}}

Segmentação de Público

Crie listas segmentadas por:

  • Tags/Etiquetas
  • Funil de vendas
  • Última interação
  • Campos personalizados

⚠️ Políticas do WhatsApp

  • Não envie spam
  • Obtenha consentimento antes de enviar
  • Ofereça opção de descadastro
  • Respeite horários comerciais
  • Não envie conteúdo proibido (violência, drogas, etc)

Troubleshooting

Mensagens não estão sendo entregues

Possíveis causas:

  • Números inválidos ou bloqueados
  • Conta do WhatsApp banida
  • Template não aprovado (API Oficial)
  • Limite de envios atingido

Solução: Verifique o status da conta e os logs de erro

Campanha pausada automaticamente

Possíveis causas:

  • Taxa de falha muito alta (>30%)
  • Muitos bloqueios de usuários
  • Limite de API atingido

Solução: Revise a lista de contatos e aguarde 24h antes de tentar novamente
