Templates de Mensagem

Crie e gerencie templates aprovados pela Meta para API Oficial

O que são Templates?

Templates são modelos de mensagem pré-aprovados pela Meta (Facebook) que você pode usar para iniciar conversas via API Oficial do WhatsApp. São obrigatórios para enviar mensagens fora da janela de 24 horas após a última interação do cliente.

Como Utilizar

Passo 1: Criar Novo Template

  1. Acesse Atendimento → Templates
  2. Clique em "+ Novo Template"
  3. Preencha os dados:
    • Nome: Identificador único (sem espaços, use underscore)
    • Categoria: Marketing, Transacional ou Autenticação
    • Idioma: pt_BR para português

Passo 2: Escrever o Conteúdo

  1. Digite o texto da mensagem
  2. Adicione variáveis usando {{1}}, {{2}}, etc
  3. Exemplo: "Olá {{1}}, seu pedido {{2}} foi enviado!"
  4. Adicione botões (opcional):
    • Call to Action: Link para site
    • Quick Reply: Respostas rápidas

Passo 3: Enviar para Aprovação

  1. Revise o template na prévia
  2. Clique em "Enviar para Aprovação"
  3. Aguarde análise da Meta (geralmente 1-24 horas)
  4. Você receberá notificação do resultado

💡 Dicas para Aprovação

  • Seja claro e objetivo no texto
  • Não use linguagem promocional excessiva
  • Sempre ofereça valor ao cliente
  • Inclua identificação da empresa
  • Evite CAPS LOCK e muitos emojis

Categorias de Templates

Marketing

Para promoções, ofertas e novidades:

  • Lançamento de produtos
  • Promoções e descontos
  • Newsletters
  • Convites para eventos

⚠️ Atenção

Templates de marketing têm aprovação mais rigorosa e podem ser rejeitados se forem muito promocionais.

Transacional

Para atualizações sobre transações:

  • Confirmação de pedido
  • Status de entrega
  • Lembretes de pagamento
  • Confirmação de agendamento

Autenticação

Para códigos de verificação:

  • Códigos OTP (One-Time Password)
  • Verificação de conta
  • Reset de senha

Variáveis Dinâmicas

Como Usar Variáveis

Variáveis permitem personalizar cada mensagem:

  • {{1}} = Primeira variável (ex: nome do cliente)
  • {{2}} = Segunda variável (ex: número do pedido)
  • {{3}} = Terceira variável (ex: data de entrega)

Exemplo de template:

Olá {{1}}! 

Seu pedido #{{2}} foi confirmado e será entregue em {{3}}.

Obrigado por comprar conosco!

Ao enviar, você preenche:

  • {{1}} = "João"
  • {{2}} = "12345"
  • {{3}} = "3 dias úteis"

Botões

Call to Action (CTA)

Botões que direcionam para ações:

  • Visitar Site: Abre URL no navegador
  • Ligar: Inicia chamada telefônica

Máximo de 2 botões CTA por template.

Quick Reply

Botões de resposta rápida:

  • Cliente clica e envia resposta predefinida
  • Útil para confirmações: "Sim", "Não", "Mais informações"
  • Máximo de 3 botões Quick Reply

Status dos Templates

Possíveis Status

  • Rascunho: Ainda não enviado para aprovação
  • Pendente: Aguardando análise da Meta
  • Aprovado: Pronto para uso
  • Rejeitado: Não passou na aprovação (veja motivo)
  • Pausado: Aprovado mas desativado temporariamente

Motivos Comuns de Rejeição

  • Conteúdo muito promocional
  • Falta de identificação da empresa
  • Gramática ou ortografia incorreta
  • Conteúdo enganoso ou sensacionalista
  • Violação das políticas do WhatsApp

⚠️ Importante

  • Templates aprovados não podem ser editados (apenas pausados/excluídos)
  • Para alterar, você precisa criar um novo template
  • Templates rejeitados podem ser editados e reenviados

Usando Templates em Campanhas

Enviar Template

  1. Acesse Atendimento → Campanhas
  2. Crie nova campanha
  3. Selecione um template aprovado
  4. Preencha as variáveis (se houver)
  5. Escolha os destinatários
  6. Envie ou agende
