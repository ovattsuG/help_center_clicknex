Como configurar a API Oficial do WhatsApp (WABA)

Guia completo para configurar e conectar a API Oficial do WhatsApp Business no CLICKNEX

Pré-requisitos

📱

Conta Business

WhatsApp Business Account ativa

🏢

Meta Business

Conta no Meta Business Manager

📞

Número Dedicado

Número de telefone exclusivo para API

Passo a Passo

1. Criar Conta no Meta Business Manager

  1. Acesse business.facebook.com
  2. Clique em "Criar conta"
  3. Preencha os dados da sua empresa
  4. Confirme seu e-mail

2. Configurar WhatsApp Business API

  1. No Meta Business Manager, acesse "WhatsApp"
  2. Clique em "Começar"
  3. Adicione seu número de telefone
  4. Verifique o número via SMS ou chamada
  5. Configure o perfil da empresa

3. Obter Credenciais de Acesso

Você precisará das seguintes informações:

  • Phone Number ID: Identificador do número
  • WhatsApp Business Account ID: ID da conta
  • Access Token: Token de acesso permanente

4. Conectar no CLICKNEX

  1. Acesse o módulo Sistema no CLICKNEX
  2. Vá em "Canais" > "Adicionar Canal"
  3. Selecione "WhatsApp Business API (Oficial)"
  4. Cole as credenciais obtidas
  5. Clique em "Conectar"
  6. Aguarde a confirmação de conexão

⚠️ Importante: Após conectar, teste enviando uma mensagem para garantir que tudo está funcionando corretamente.

💡 Dicas de Configuração

  • Use um número dedicado exclusivamente para a API
  • Configure webhooks para receber mensagens em tempo real
  • Ative a verificação em duas etapas para segurança
  • Mantenha o token de acesso seguro
