O que é o Cadastro Incorporado?

O Cadastro Incorporado (Embedded Signup) é uma funcionalidade que permite criar e configurar uma conta WhatsApp Business API diretamente pelo CLICKNEX, sem precisar acessar o Meta Business Manager.

Coexistência

Coexistência de APIs

A funcionalidade de coexistência permite que você use simultaneamente a API Oficial (WABA) e APIs Não Oficiais (Baileys, Evolution) no mesmo sistema, com números diferentes.

Casos de Uso

  • Migração Gradual: Migre de API não oficial para oficial aos poucos
  • Teste A/B: Compare performance entre APIs
  • Múltiplos Números: Use API oficial para número principal e não oficial para testes
  • Backup: Mantenha canal alternativo em caso de problemas

💡 Recomendações

  • Use API Oficial para operações críticas
  • Monitore custos da API Oficial regularmente
  • Documente qual número usa qual API
  • Planeje migração completa para API Oficial
