Cadastros - Clientes Financeiros

Gerencie todos os clientes vinculados às suas operações financeiras

Neste módulo, você gerencia todos os clientes vinculados às suas operações financeiras no CLICKNEX. Centralizar essas informações é crucial para a emissão correta de faturas, acompanhamento de inadimplência e histórico de transações.

Visão Geral da Lista de Clientes

A tela principal exibe uma tabela detalhada com todos os seus clientes cadastrados. Você pode gerenciar sua visualização e encontrar informações rapidamente:

Barra de Pesquisa

Utilize para buscar rapidamente por nome, documento ou email do cliente.

Filtros Avançados

Refine sua busca usando os dropdowns de Tipo (Pessoa Física/Jurídica) e Status (ex: Ativo).

Tabela de Dados

Visualize todas as informações essenciais dos clientes em colunas organizadas.

Ações

Menu com opções adicionais para gerenciar o registro do cliente.

Colunas da Tabela

  • Nome: Nome completo ou razão social
  • Tipo: Classificação como Pessoa Física ou Pessoa Jurídica
  • Documento: CPF ou CNPJ (exibido como '-' se não preenchido)
  • Email: O endereço de contato principal
  • Telefone: O número de telefone de contato principal
  • Status: O status atual do cliente (ex: Ativo)
  • ... (Ações): Menu com opções adicionais para gerenciar o registro

Paginação

Controle quantos registros são exibidos por página e navegue entre elas.

Como Adicionar Novos Clientes

Existem duas formas principais de adicionar novos clientes ao seu módulo financeiro:

1. Importar do CRM

Clique no botão [Importar do CRM] (azul, no topo direito) para sincronizar clientes já cadastrados no seu módulo de CRM principal. Isso evita duplicidade de dados e agiliza o processo.

2. Cadastrar Novo Cliente Manualmente

Clique no botão [+ Novo Cliente] (verde, no topo direito) para abrir o formulário de cadastro.

Campos do Formulário

Preencha os seguintes campos no modal que se abre:

Tipo (Obrigatório)

Selecione se o cliente é uma Pessoa Física ou Pessoa Jurídica. O formulário se ajustará dinamicamente.

CNPJ / CPF (Obrigatório)

Insira o número do documento com a máscara correta.

Nome (Obrigatório)

Insira o nome completo ou a razão social.

Email

O email principal para faturamento e contato.

Telefone

O número de telefone principal de contato.

Endereço

Inclua rua, número, complemento, cidade, estado (UF) e CEP.

Observações

Espaço opcional para adicionar informações relevantes sobre o cliente.

Após preencher os dados, clique em [Criar] para salvar o novo cliente.

Importante: Lembre-se de verificar a precisão dos dados inseridos, especialmente documento e endereço de faturamento, para garantir a conformidade legal na emissão de documentos financeiros.

Dicas

  • Use a importação do CRM para evitar duplicidade de dados
  • Mantenha os dados de contato sempre atualizados
  • Preencha o endereço completo para facilitar a emissão de notas fiscais
  • Use o campo de observações para informações importantes sobre o cliente
