Configurações do WhatsApp

Configure conexões e parâmetros do WhatsApp

O que são Configurações do WhatsApp?

Aqui você gerencia as conexões com WhatsApp, tanto API Oficial quanto Não Oficial, configurando números, webhooks e parâmetros de envio.

Como Utilizar

Conectar Número

  1. Acesse Sistema → Config. WhatsApp
  2. Clique em "+ Adicionar Número"
  3. Escolha o tipo: API Oficial ou Não Oficial
  4. Siga o processo de autenticação
  5. Aguarde validação

Configurações Disponíveis

  • Mensagem de ausência automática
  • Horário de atendimento
  • Limite de mensagens por minuto
  • Webhooks para integrações
  • Backup de conversas

⚠️ Importante

Não desconecte o número durante campanhas ativas. Isso pode causar falhas nos envios.
