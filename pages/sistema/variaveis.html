Variáveis Personalizadas

Crie campos customizados para armazenar informações específicas

O que são Variáveis?

Variáveis personalizadas são campos adicionais que você pode criar para armazenar informações específicas do seu negócio em leads, contatos e negócios.

Como Utilizar

Criar Variável

  1. Acesse Sistema → Variáveis
  2. Clique em "+ Nova Variável"
  3. Defina:
    • Nome: Ex: "Tamanho da Empresa"
    • Tipo: Texto, Número, Data, Lista
    • Onde usar: Leads, Contatos, Negócios
  4. Salve

Tipos de Variáveis

  • Texto: Campo livre
  • Número: Valores numéricos
  • Data: Datas específicas
  • Lista: Opções pré-definidas
  • Sim/Não: Checkbox

💡 Exemplos de Uso

  • Número de funcionários
  • Segmento de atuação
  • Data de aniversário
  • Preferência de contato
