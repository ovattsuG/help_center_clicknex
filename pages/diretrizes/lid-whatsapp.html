LID do WhatsApp

Entenda o que é o LID e como gerenciar contatos duplicados

O que é o LID?

O LID é um identificador temporário que o WhatsApp usa para novas conversas, a fim de proteger a privacidade do usuário.

Por que acontece?

Quando um cliente que não está na sua lista de contatos inicia uma conversa, o WhatsApp pode enviar primeiro este LID em vez do número de telefone real.

📱 Comportamento do Sistema

O CLICKNEX exibe o LID que recebe. Assim que o cliente interage e o WhatsApp libera o número real, o sistema automaticamente atualiza o contato.

⚠️ Limitação Temporária

Enquanto o sistema só tiver o LID, não é possível responder à conversa, pois a plataforma ainda não tem o número de telefone.

Como Resolver Contatos Duplicados

Recomendado

Método 1: Agrupamento na Tela de Atendimento

Forma mais rápida de resolver uma duplicação identificada:

1. Dentro da tela de atendimento do contato com LID

2. Localize a opção "@lid Agrupar Contato"

3. O sistema fará a fusão automática das conversas

Alternativo

Método 2: Agrupamento na Página de Contatos

Útil para fazer limpeza geral na base de contatos:

1. Acesse a página "Contatos"

2. Selecione o contato a corrigir

3. Use a opção de agrupar/atualizar LID

💡 API Oficial e o LID

O problema de duplicação por LID é característico de APIs não oficiais. Esta situação não acontece ao utilizar a API Oficial do WhatsApp (WABA), que sempre utiliza o número de telefone como identificador único.
